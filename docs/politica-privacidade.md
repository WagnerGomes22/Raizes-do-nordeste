# Política de Privacidade

Última atualização: 25/06/2026

## 1. Escopo

Esta política descreve como a aplicação `Raízes do Nordeste` trata dados pessoais no contexto atual do sistema, que hoje opera como fluxo visual com persistência local no navegador.

## 2. Dados tratados

O sistema pode tratar os seguintes dados:

- nome;
- email;
- telefone;
- endereço;
- preferências de perfil;
- itens do carrinho;
- status do pedido;
- preferências relacionadas a cookies e navegação.

## 3. Finalidades

Os dados são utilizados para:

- viabilizar login e cadastro em fluxo visual;
- manter o carrinho persistido;
- simular pedido, pagamento e fidelidade;
- exibir preferências e informações do perfil;
- melhorar continuidade de navegação no navegador local.

## 4. Base operacional atual

No estágio atual do projeto, os dados são armazenados em `LocalStorage` para fins de demonstração e experiência local. Não há backend real nem compartilhamento com serviços externos na navegação principal implementada.

## 5. Direitos do usuário

O usuário pode:

- revisar seus dados visíveis no perfil;
- alterar preferências;
- apagar os dados locais limpando o armazenamento do navegador;
- interromper o uso da aplicação a qualquer momento.

## 6. Segurança e minimização

O projeto busca seguir princípios de minimização e transparência:

- coletar apenas dados necessários ao fluxo demonstrativo;
- evitar exposição de segredos no código;
- manter dados sensíveis fora do repositório;
- preparar o sistema para evolução segura futura.

## 7. Atualizações

Este documento deve ser revisado sempre que o sistema passar a contar com backend real, autenticação real, integrações com pagamentos, analytics ou serviços de terceiros.
